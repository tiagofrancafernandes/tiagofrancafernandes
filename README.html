About me

I am a Senior Backend Engineer from Brazil, focused on building scalable, maintainable and API-first systems with PHP, Laravel, PostgreSQL, Docker, Linux and AWS.

My strongest background is in backend engineering, API architecture, database performance, third-party integrations and legacy system modernization. I also work comfortably with Vue.js, Nuxt and Tailwind CSS, especially when building frontend integrations for backend-driven products.

I have worked with SaaS platforms, internal enterprise systems, automotive platforms, business tools, financial/business integrations and custom web applications. My work usually involves designing APIs, improving architecture, optimizing queries, refactoring legacy code, creating reliable development environments and helping teams make better technical decisions.

I also work as a programming mentor, teaching PHP, Laravel, Linux, Docker and software engineering fundamentals.
